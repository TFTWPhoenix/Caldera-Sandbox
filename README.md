# Caldera-Sandbox

Toolkit for making a game in Minecraft.

### Requirements
- [x] Spigot (or derivatives) 1.8.8 server
- [x] Java 1.8

### Features
- [x] Stats System
- [x] Custom Items
- [x] Item Abilities
- [x] Custom Mobs
- [x] Command-based Item Editor (GUI-based version & improved CLI-based version coming soon)
